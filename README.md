# Restoran

## Features

- **User  Authentication**: Secure login and registration for customers and restaurant staff.
- **Cart Functionality**: Customers can add items to their cart, view their cart, and modify item quantities before checkout.
- **Order Management**: Customers can place orders.
- **Responsive Design**: A user-friendly interface that works well on both desktop.

## Technologies Used

- **Backend**: Java, Spring Boot, Spring Security, JPA/Hibernate
- **Database**: Oracle
- **Frontend**: Angular
- **Build Tool**: Maven 
- **Version Control**: Git
